SimplePromises.js

Simple-Promises is a super-bare-bones implementation of the Promises/A specification proposal, written mainly as a learning exerciseâ€”there are better implementations out there, suitable for real-world use.

I have heavily commented the code as it took me far longer to figure this out than it should, and I have a pernicious habit of forgetting the subtleties of such code. Hopefully, these comments might also provide insight to others.

The ELI5 Version

Promises allow us to write asynchronous code to appear more synchronous, leading to code that is easier to read and write. They effectively prevent the pyramid of doom!

Example

Simple-Promises allow us to write our asynchronous code in the form:

foobar()
    .then(foo)
    .then(bar)
    .then(function () {
        return baz();    // Where `baz` is a predefined promise instance
    })
    .then(function () {
        return fooBaz();
    })
    .then(bar);

Where foobar, baz and fooBaz are predefined Promise instances.
