Development

SimplePromises.js

A straightfoward, partial implementation of the Promises/A+ specification. I wrote this before Promises were part of the ECMAScript 6 specification, as a challenge, and a learning opportunity. It was a worthwhile experiment, though should not be considered production ready.

Grids

Grids is a small drop-in solution to render interactive SVG grids graphs, charts and layout grids. Designed for performance and extensibility, this small library requires no external dependencies. The solution was built as a self-contained module, originally created as part of a larger SVG graphing solution.

Vandegraaff

Conceived and developed to power this site, Vandegraaff is a versatile, light-weight and cross-platform site generator written in C# for ASP.NET 5. It can be configured to output static content, or dynamically render markdown in real-time.

hMVC

hMVC is a hierachical MVC JavaScript framework. Each view can contain child views, each with their own discrete contexts. At its core it uses Object.Observe to implement bi-directional data-binding.
